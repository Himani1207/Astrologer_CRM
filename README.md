# Humara Pandit CRM Dashboard

A web-based Customer Relationship Management (CRM) dashboard designed for managing the Humara Pandit platform. The system enables administrators to manage astrologers, customers, consultations, payments, and platform operations through a centralized and user-friendly interface.

## Features

### Dashboard

- View key business metrics and platform statistics
- Monitor revenue and consultation trends
- Track recent consultations and activities
- Analyze astrologer performance

### Astrologer Management

- Add, update, and manage astrologer profiles
- Monitor online and offline status
- Track performance and consultation activity

### Customer Management

- Maintain customer records
- View customer details and contact information
- Access consultation history

### Consultation Management

- Track consultation bookings and records
- Monitor consultation duration and status
- Manage payment and booking information

### Analytics & Reports

- Visualize revenue trends and earnings
- Monitor consultation activity
- Track active users and platform engagement
- Interactive charts and reports powered by Chart.js

### Settings & Access Control

- Manage administrator profiles
- Configure system settings
- Manage user roles and permissions

## Tech Stack

- React 19
- Vite
- JavaScript (ES6+)
- Vanilla CSS
- Lucide React
- Chart.js
- React Chart.js 2

## Installation

Install project dependencies:

```bash
npm install
```

## Running the Application

Start the development server:

```bash
npm run dev
```

Build the application for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Run ESLint:

```bash
npm run lint
```

## Purpose

The Humara Pandit CRM Dashboard streamlines the management of astrologers, customers, consultations, and business analytics, helping administrators efficiently oversee platform operations from a single dashboard.
